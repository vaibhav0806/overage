# Overage (overage.app)

  ## What this is
  A web app that helps freelancers track project scope drift, calculate unbilled
  dollar value of additions, and generate professional scope change reports to
  send to clients.

  ## Key decisions (already validated)
  - Blue ocean — no direct competitor does this
  - Target: freelance devs, designers, consultants, small agencies
  - Pricing: Free 1 project, $8/mo unlimited
  - Distribution: client-facing reports have branding = organic exposure
  - Tech: Next.js + NeonDB + Drizzle + Better Auth + DodoPayments + Railway
